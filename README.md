How to run the project

1. Clone the Project or Download it
2. open backend folder and install all the dependencies using npm install [package name]
3. run command - node app.js (dont close the terminal)
4. open frontend and install all the dependencies
5. run command - npm start


User Features

1. Browse Hotels & Resorts — Users can explore listings with images, descriptions, prices, and amenities.
2. Search & Filter — Find properties by location, price range, ratings, or keywords.
3. View Details Page — Each listing displays full details, user reviews, and available booking dates.
4. Book Hotels/Resorts — Real-time reservation system for easy and secure bookings.
5. Leave Reviews & Ratings — Users can share experiences and rate properties after stays.
6. Authentication System — Secure user registration and login with JWT-based authentication.
7. Profile Management — View and manage booking history, personal info, and reviews.

Technologies used: HTML, CSS, Bootstrap, JavaScript, React.js, Node.js, Express.js, MongoDB, RESTful APIs, JWT.
